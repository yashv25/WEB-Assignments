16. What is the purpose of the <head> tag in an HTML document? Answer: The <head> tag contains meta-information about the document, including its title, character set, styles, and links to external resources like CSS and JavaScript files. The content within the <head> tag is not visible on the webpage.

19. What is the purpose of the alt attribute in the <img> tag? Answer: The alt attribute provides alternative text for an image if the image fails to load or if the user is using a screen reader. This improves accessibility and helps search engines understand the content of the image.
